# Project Overview

This project contains a Python script for analyzing Confluence spaces and generating HTML reports. It also includes related documentation about AI-assisted development frameworks.

# Key Files

*   `manage-spaces/confluence_space_report.py`: A Python script that connects to the Confluence API, fetches all content from a specified space, categorizes it by age and author, and generates a detailed HTML report.
*   `docs/AIDevelopmentFramework.html`: A document outlining a framework for leveraging AI in the development process, focusing on shifting the developer's role from implementation to architectural oversight.
*   `manage-spaces/confluence_DEV_content_analysis.html`: An example HTML report generated by the `confluence_space_report.py` script for the "DEV" space in Confluence.

# Building and Running

The `confluence_space_report.py` script can be run from the command line.

**Prerequisites:**

*   Python 3
*   The `requests` and `jinja2` libraries. These can be installed using pip:
    ```bash
    pip install requests jinja2
    ```

**Running the script:**

To run the script and generate a report for the default "DEV" space, navigate to the `manage-spaces` directory and execute the following command:

```bash
python confluence_space_report.py
```

To specify a different Confluence space, pass the space key as a command-line argument:

```bash
python confluence_space_report.py <SPACE_KEY>
```

The generated HTML report will be saved in the current directory.

# Development Conventions

*   The Python script includes a configuration section at the top where the user's email, API token, and default Confluence space can be set.
*   **Security Note:** The script currently stores the Confluence API token directly in the source code. For improved security, it is recommended to use environment variables or a more secure method for managing secrets.
*   The script uses the `jinja2` templating engine to generate the HTML report, which allows for a clean separation of data and presentation.
