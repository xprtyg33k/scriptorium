# Scriptorium Project Overview

## Project Type
This is a comprehensive documentation and content management repository that contains:
1.  A collection of HTML-based documents and infographics on AI development frameworks.
2.  A Python script for generating Confluence space content reports.
3.  A suite of tools for the FOSDEM 2026 conference schedule.

## Directory Structure
*   `/docs`: Contains HTML documents, infographics, and resources on AI development topics.
*   `/manage-spaces`: Contains a Python script for Confluence content analysis and reporting.
*   `/fosdem`: Contains a set of scripts and a web-based planner for the FOSDEM 2026 conference.

## Key Components

### Documentation Portal (`/docs`)
A self-contained HTML-based documentation portal with:
*   `index.html`: Main entry point that dynamically lists available documents.
*   `manifest.json`: Lists all available HTML documents (e.g., `AIDevelopmentFramework.html`, `advr.html`, `BMAD-infographic.html`).
*   `paletteConverter.html`: A color palette converter tool.

The documentation uses Tailwind CSS for styling and Chart.js for data visualizations. The portal is designed to be hosted statically and fetches document lists from the manifest file.

### Confluence Management (`/manage-spaces`)
*   `confluence_space_report.py`: A Python script that connects to the Confluence REST API, fetches all content from a specified space, categorizes it by age and author, and generates a detailed HTML report.
*   `confluence_DEV_content_analysis.html`: An example HTML report generated by the script.

**Features:**
*   Classifies content by age (YTD, 1 year, 2 years, etc.).
*   Identifies content created by deprecated accounts.
*   Uses `jinja2` templating for report generation.

**Security Note:** The script currently stores the Confluence API token directly in the source code. For improved security, it is recommended to use environment variables or a more secure method for managing secrets.

### FOSDEM Scheduler (`/fosdem`)
A suite of tools and a web application for planning FOSDEM 2026 attendance.
*   **Web Planner:** An interactive, single-file HTML application (`index.html`) for searching, rating, and scheduling talks. Features conflict detection, local storage persistence, and export capabilities.
*   **Data Processing:** Python scripts (e.g., `parse_xml.py`, `update_html.py`) to parse the official FOSDEM XML schedule and generate JSON data.

## Technologies Used
*   **Documentation:** HTML/CSS/JavaScript, Tailwind CSS, Chart.js.
*   **Confluence Scripts:** Python (`requests`, `jinja2`).
*   **FOSDEM App:** Vanilla JavaScript, HTML5, CSS Grid/Flexbox.

## Development Conventions
*   **Python:** Follows standard conventions with configuration sections. Ensure dependencies are installed via `pip`.
*   **HTML:** Documents follow consistent styling. `fosdem` tools emphasize single-file portability where possible.
*   **License:** All content is released under the Unlicense (public domain).

## Building and Running

### Documentation Portal
Serve locally with any HTTP server:
```bash
python -m http.server 8000 --directory docs
# Open http://localhost:8000
```

### Confluence Report Generator
1.  Install dependencies:
    ```bash
    pip install requests jinja2
    ```
2.  Run the script (ensure API credentials are configured):
    ```bash
    python manage-spaces/confluence_space_report.py [SPACE_KEY]
    ```

### FOSDEM Planner
1.  Navigate to the directory:
    ```bash
    cd fosdem
    ```
2.  Serve the application:
    ```bash
    python -m http.server 8000
    # Open http://localhost:8000/index.html
    ```
